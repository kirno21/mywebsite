Pemrograman Web 1

TIF-301

Semester 5 • 3 SKS

Mata kuliah Pemrograman Web 1 membahas dasar-dasar pengembangan aplikasi berbasis web, mencakup konsep struktur halaman web menggunakan HTML, tata letak dan desain dengan CSS, serta interaktivitas sederhana menggunakan JavaScript.

Topik Pembelajaran:

  • Pengenalan Teknologi Web
  • Dasar-dasar HTML
  • Formulir HTML
  • Dasar-dasar CSS
  • Layout dengan CSS
  • Pengenalan JavaScript
  • Integrasi HTML, CSS, dan JavaScript
  • Publikasi Website Sederhana

Tentang Saya

Saya adalah mahasiswa semester 5 Program Studi Teknik Informatika di Universitas Pamulang. Saat ini saya sedang mendalami berbagai mata kuliah lanjutan di bidang pemrograman, basis data, dan pengembangan aplikasi berbasis web. Saya memiliki ketertarikan kuat pada dunia teknologi, khususnya software development dan pemrograman web, serta terus berusaha meningkatkan keterampilan di bidang ini melalui tugas kuliah, proyek mandiri, maupun kegiatan organisasi kampus.


Saya juga aktif mengikuti pelatihan dan seminar teknologi untuk menambah wawasan di luar materi perkuliahan, sekaligus memperluas jejaring profesional. Dengan bekal kemampuan analisis, pemecahan masalah, dan kerja sama tim, saya optimis dapat memberikan kontribusi positif dalam pengembangan teknologi informasi, baik di lingkungan akademik maupun di dunia kerja nantinya.
